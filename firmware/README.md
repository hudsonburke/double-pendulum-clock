This is where microcontroller code will live

I highly recommend using PlatformIO for development, as it provides a great environment for embedded development and supports a wide range of microcontrollers.

You can install PlatformIO as an extension in Visual Studio Code, which will give you access to a powerful IDE for writing and debugging your firmware.

[Instructions and documentation](https://docs.platformio.org/en/latest/integration/ide/vscode.html)
